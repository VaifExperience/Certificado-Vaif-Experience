CERTIFICADO VAIF EXPERIENCE - GitHub Pages

Como publicar:
1. Entre no seu repositório no GitHub.
2. Clique em Add file > Upload files.
3. Envie o arquivo index.html desta pasta.
4. Clique em Commit changes.
5. Vá em Settings > Pages.
6. Em Source, escolha Deploy from a branch.
7. Branch: main | Folder: /root.
8. Clique em Save.

Depois o GitHub vai gerar um link parecido com:
https://vaifexperience.github.io/Certificado-Vaif-Experience/

O site contém:
- formulário para digitar o nome da participante;
- certificado gerado automaticamente;
- botão de imprimir/salvar em PDF;
- link compartilhável com o nome preenchido.
